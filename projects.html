Braille Translator

The Braille Translator is a software application that uses Unified English Braille orthography rules to translate text into Braille. It is written in Python and uses Tkinter for the graphical user interface. The translator is capable of handling a wide range of text, including numbers, punctuation, and special characters. With the Braille Translator, users can easily and accurately translate text into Braille, making it accessible to visually impaired individuals.

Egg Game

The Egg Game showcases easy-to-read Python classes and loops with Turtle graphics. The rules are easy: How many red eggs can the user drag into the basket within 60 seconds? However, only the red eggs are allowed in the basket. To create a red egg, catch the white square with an egg that is NOT red. If the white square catches a red egg before it's placed in the basket, it's game over.
